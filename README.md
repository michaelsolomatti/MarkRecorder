🎙️ MarkRecorder — диктофон с умными закладками

> Профессиональный диктофон с возможностью ставить метки (закладки) во время записи и при воспроизведении.  
> Идеально для записи лекций, интервью, совещаний и аудиозаметок с быстрым доступом к ключевым моментам.  
> Реализован на 7 языках программирования.

## 🧠 Концепция

**MarkRecorder** — это не просто диктофон. Это инструмент для структурирования аудиоинформации:

- ✅ **Запись звука** с микрофона в формате WAV (с возможностью выбора качества).
- ✅ **Закладки (метки)** — во время записи или воспроизведения можно поставить метку с комментарием.
- ✅ **Хранение меток** в JSON-файле рядом с аудио.
- ✅ **Воспроизведение** с перемоткой к меткам (в GUI – по клику, в консоли – по номеру).
- ✅ **Список записей** — отображение всех сохранённых файлов с датами и длительностью.
- ✅ **Визуализация** звука (в GUI-версиях — осциллограмма / спектрограмма).
- ✅ **Автоопределение тишины** (опционально) — автоматическое добавление меток на паузы.
- ✅ **Экспорт** меток в текстовый файл.
- ✅ **Горячие клавиши** (пробел — старт/стоп, M — метка, P — воспроизведение).

## 🚀 Функциональность

| Действие                  | Описание |
|---------------------------|----------|
| `Новая запись`            | Начать запись с микрофона. |
| `Остановить запись`       | Завершить запись, сохранить аудио и метки. |
| `Поставить метку`         | Добавить метку в текущую позицию (время от начала). |
| `Воспроизвести`           | Проиграть выбранную запись. |
| `Перейти к метке`         | В GUI — клик по метке, в консоли — команда `goto <id>`. |
| `Список записей`          | Показать все записи с метаданными. |
| `Удалить запись`          | Удалить аудиофайл и файл меток. |
| `Экспорт меток`           | Сохранить метки в текстовый файл для отчёта. |

## 📦 Содержимое репозитория

| Файл                  | Язык       | Особенности |
|-----------------------|------------|-------------|
| `recorder_python.py`  | Python     | + Tkinter GUI, `pyaudio` + `wave`, визуализация через `matplotlib` (опционально) |
| `recorder_cpp.cpp`    | C++        | + Qt Widgets, `QAudioInput`, `QAudioOutput`, таблица меток |
| `recorder_java.java`  | Java       | + Swing, `TargetDataLine`, `SourceDataLine`, таблица меток |
| `recorder_cs.cs`      | C#         | + WPF, `NAudio`, график звука через `OxyPlot` (опционально) |
| `recorder_go.go`      | Go         | + консоль, запись через `arecord` (Linux) или `sox`, управление командами |
| `recorder_rs.rs`      | Rust       | + консоль, запись через `ffmpeg`/`sox`, цветной вывод |
| `recorder_js.js`      | JavaScript | + Electron, Web Audio API, MediaRecorder, визуализация на Canvas |

## 🛠️ Как запустить

Каждая версия требует установки соответствующих библиотек/инструментов. Инструкции:

```bash
# Python
pip install pyaudio numpy matplotlib  # для визуализации
python recorder_python.py

# C++ (Qt)
qmake && make
./recorder_cpp

# Java
javac -cp . recorder_java.java && java recorder_java

# C# (WPF)
dotnet add package NAudio
dotnet run

# Go (требуется arecord или sox)
go mod tidy
go run recorder_go.go

# Rust (требуется ffmpeg или sox)
cargo build --release && ./target/release/recorder_rs

# Node.js (Electron)
npm install
npm start
